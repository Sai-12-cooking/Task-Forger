# 📋 Task Forge

Task Forge is a modern, feature-rich productivity operating system built with React Native and Expo. It combines intelligent task prioritization, habit tracking, and focus management into a single, unified experience that works seamlessly across Web, iOS, and Android.

## ✨ Features

- 🎯 **Eisenhower Matrix**: Prioritize your tasks effortlessly by categorizing them into Urgent & Important quadrants.
- 🔁 **Recurring Tasks**: Automate your workflow with tasks that repeat daily, weekly, on weekdays, or monthly.
- ⏰ **Smart Reminders**: Cross-platform notifications ensure you never miss a deadline.
- 🍅 **Focus Mode**: Built-in Pomodoro timer to track deep work sessions and tie them directly to your tasks and habits.
- 🌱 **Habit Tracking**: Build and maintain daily routines with detailed streak tracking and completion logs.
- 📅 **Agenda Planner**: A visual calendar interface to plan your days and weeks ahead.
- 📊 **Insights & Stats**: Visualize your productivity trends, focus hours, and completion rates over time.
- ⌨️ **Keyboard Shortcuts**: Power-user friendly design for quick navigation and task creation.

## 🛠️ Tech Stack

- **Framework**: React Native with Expo
- **Storage**: AsyncStorage for local-first data persistence
- **Notifications**: Expo Notifications (with custom Web API fallback support)
- **Icons & UI**: Custom dynamic theming system

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sai-12-cooking/Task-Forger.git
   cd Task-Forger
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npx expo start
   ```

### Running on different platforms:
- **Web**: Press `w` in the terminal to open it in your browser.
- **Android**: Press `a` (requires Android Studio emulator or Expo Go app on your physical device).
- **iOS**: Press `i` (requires Xcode simulator on macOS or Expo Go app on your physical device).
